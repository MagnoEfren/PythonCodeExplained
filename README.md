## capitalize()

capitalize
Ejemplo: cadena = "hola mundo", cadena.capitalize() devuelve "Hola mundo".

```python
cadena = "hola mundo"
cadena.capitalize()
Ouput:
"Hola mundo"
```
Explicación:
Devuelve una copia de la cadena con la primera letra en mayúscula y el resto en minúscula.
